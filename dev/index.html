mijn vaardigheden

responsive design

Ik design en codeer altijd met de gedachte mobile first. In het hedendaagse leven is een website die goed werkt op alle devices een pré.
Het zorgen voor een fijne User Experience voor de gebruiker op elk schermformaat is ook iets waar ik op focus.

coderen

Ik bouw website’s met html5, css3 en javascript. Ik hou ervan om van de grond op iets te bouwen, maar kan ook omgaan met frameworks zoals bootstrap en jQuery.
Bij het schrijven van code focus ik op 3 dingen Clean, performance & cross browser support

UX

Ik ontwerp altijd met het goal om de beste UX voor de gebruiker te leveren. Een duidelijke flow creëeren in mijn ontwerpen is altijd van belang.
Zorgen dat mijn code ook altijd geoptimaliseerd is voor performance zodat de gebruiker een betere ervaring heeft is ook altijd een pré bij mij.
